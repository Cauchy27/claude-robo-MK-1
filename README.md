# 🤖 claude-robo-MK-1

日本語ロボ口調プラグイン。Claude Code の出力トークンを **30-50% 削減**する。

> Before:「ファイルを確認しました。バグが3つほど見つかったようです。修正しますね。」
> After: 「ファイル確認済。バグ3件検出。修正スル。」

## 特徴

- 日本語特化（敬語・フィラー・助詞の冗長性を削減）
- 技術的正確性は完全維持（コード・API名・ビジネスロジック保護）
- SessionStart で自動有効化
- ロボット口調でキャラクター付け（🤖）

## レベル切替（slash commands）

```
/robo-lite      # 20-30%削減（安全・文脈最優先）
/robo-normal    # 30-50%削減（推奨デフォルト）
/robo-ultra     # 50-70%削減（短文・上級者向け）
/robo-off       # 通常日本語に戻す
/robo-status    # 現在モード確認
```

## 設計原則

**文脈保持 > トークン削減**

削減と明確性が衝突したら、必ず明確性を優先する。技術的指示・ビジネスロジック・コード内容は一切圧縮しない。

## インストール

### 1. マーケットプレイス追加

```bash
/plugin marketplace add Cauchy27/claude-robo-MK-1
```

### 2. プラグインインストール

```bash
/plugin install claude-robo-MK-1@claude-robo-MK-1
```

### 3. セッション再起動

Claude Code を再起動すると次のバナーが表示される:

```
🤖 claude-robo-MK-1 起動シマス
```

## 無効化

```
通常モードに戻して
```

と発言すれば即座に通常応答に戻る。

完全アンインストールは:

```bash
/plugin uninstall claude-robo-MK-1
```

## 削減メカニズム

| 削減対象 | 例 |
|---------|-----|
| 敬語 | 「〜です」→「〜ダ」 |
| フィラー | 「ちなみに」「とりあえず」→削除 |
| 前置き | 「承知しました」→削除 |
| 冗長修飾 | 「少し」「非常に」→削除 |
| 曖昧表現 | 「〜かもしれません」→「〜可能性アリ」 |

## 保持対象

- コードブロック内の全文字
- 技術用語・API名・ファイルパス
- 数値・固有名詞・エラーメッセージ
- ビジネスロジック
